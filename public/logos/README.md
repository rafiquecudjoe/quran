# Ismail Academy Logos

Please save your logo images in this directory with the following names:

## Primary Logo
- **Filename**: `ismail-academy-logo.png`
- **Description**: Main logo with shield, minaret, and "ISMAIL ACADEMY" text
- **Recommended size**: 200x200px or higher (square format)
- **Usage**: Header, main branding

## Logo with Arabic Text (Optional)
- **Filename**: `ismail-academy-logo-arabic.png`
- **Description**: Logo version with Arabic text above
- **Recommended size**: 200x200px or higher
- **Usage**: Special occasions, formal documents

## Logo Usage Guidelines

### Colors Used in Website Theme:
- **Primary Blue**: #1d4ed8 (blue-700)
- **Deep Blue**: #1e3a8a (blue-800) 
- **Dark Blue**: #1e40af (blue-900)
- **Light Blue**: #dbeafe (blue-100)
- **Accent Green**: #065f46 (matching your logo banner)

The website theme has been updated to match your beautiful logo's color scheme!

### Instructions:
1. Save your logo files in this directory
2. The website will automatically use them in the header
3. If the logo image fails to load, it will fall back to the book icon
